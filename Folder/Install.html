Runtime

The freely-available Java Runtime Environment (JRE) must be installed on the computer. JRE Version 6 (32 or 64 bit) or newer is required. The 64 bit version is preferred. The currently installed version of the JRE can be determined from the Command Prompt as shown here:

> java -version
java version "1.6.0_31"
Java(TM) SE Runtime Environment (build 1.6.0_31-b05)
Java HotSpot(TM) Client VM (build 20.6-b01, mixed mode, sharing)

If the displayed version is at least 1.6.0 (which indicates JRE 6), no Java installation is necessary. Otherwise, a newer JRE must be installed first. JRE installation requires Windows administrative rights.
